## Overview

Number Mastermind is a number guessing game developed using HTML, CSS, JavaScript, and Python. The objective of the game is to guess a secret 4-digit number with unique digits within 10 attempts. The game provides feedback after every guess to help players identify the correct number.

---

## Features

* Random 4-digit secret number generation
* Unique digit validation
* Maximum 10 attempts
* Real-time input validation
* Feedback system for guesses
* Progress bar for attempts tracking
* Responsive user interface
* Restart game functionality

---

---

## How the Game Works

1. The system generates a random 4-digit secret number.
2. The player enters a 4-digit guess.
3. The game checks:

   * Correct digits in the correct position
   * Correct digits in the wrong position
4. Feedback is displayed after every attempt.
5. The player wins by guessing the correct number within 10 attempts.
